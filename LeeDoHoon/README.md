# TMDB 데이터 EDA 분석 프로젝트

## 📋 프로젝트 개요

이 프로젝트는 TMDB(The Movie Database) 데이터를 활용하여 OTT 플랫폼별 작품 특성과 숨은 명작을 분석하는 탐색적 데이터 분석(EDA) 프로젝트입니다.

### 분석 목적

1. **OTT별 작품 필터링 기능** → Provider별 장르 강세 분석
2. **숨은 명작 찾기** → 투표수 낮고 평점 높은 작품 분석
   - 조건: 투표수는 평균 이상, 사분위수 90% 이상, 로튼 6점대 이상

### 데이터 출처

- **데이터 파일**: `data/tmdb_combined_10k.csv`
- **데이터 구성**: 총 7개 컬럼, 약 10,000개 행
- **주요 컬럼**: 
  - `id`: 작품 고유 ID
  - `title`: 작품 제목
  - `type`: 작품 유형 (movie/tv)
  - `genres`: 장르 (쉼표로 구분)
  - `providers`: OTT 플랫폼 (쉼표로 구분)
  - `vote_count`: 투표수
  - `vote_average`: 평균 평점

1. 🌐 OTT별 작품 필터링 및 Provider별 장르 강세 분석
분석 목적
각 OTT 플랫폼(Provider)이 어떤 장르의 작품에 강점을 가지고 있는지 파악합니다.

이는 사용자가 특정 장르의 작품을 찾을 때 어떤 플랫폼을 선택해야 하는지 추천하는 근거가 되며, 플랫폼의 콘텐츠 포트폴리오 전략을 이해하는 데 도움을 줍니다.

분석 과정 및 결과 설명
주요 OTT 플랫폼별 작품 수 분석:

먼저, Watcha, Netflix, TVING, Disney Plus, wavve 등 주요 플랫폼별로 보유 작품 수를 확인합니다. (예: Watcha 1,129개, Netflix 965개 등)

플랫폼별 장르 강세 분석:

각 플랫폼이 보유한 전체 작품 중 특정 장르가 차지하는 비율을 계산하여 '대표 장르'를 선정합니다.

주요 결과 예시 (자료 활용):

Netflix: 드라마 (54.1%)

Disney Plus: 코미디 (39.4%)

Watcha/wavve/TVING: 모두 드라마가 47%~55%대로 가장 높은 비중을 차지했습니다.

시각화 활용:

**플랫폼별 작품 수 막대 그래프 (03_provider_count.jpg)**로 플랫폼 크기를 보여주고,

플랫폼별 장르 TOP 5 그래프 (04_provider_genre_top5.jpg) 또는 **히트맵 (05_provider_genre_heatmap.jpg)**을 활용하여 시각적으로 강세 장르를 강조합니다.

💡 핵심 인사이트

2. 💎 숨은 명작 찾기 분석 (Hidden Gems)
분석 목적
투표수는 낮아 대중적인 인지도는 떨어지지만, 평점은 높아 전문가나 소수 관객에게 호평받는 '숨은 보석(Hidden Gems)' 같은 작품을 발굴하여 추천합니다.

분석 전략 및 조건 (정확하게 설명해야 할 부분)
숨은 명작을 정의하기 위한 4가지 까다로운 조건을 적용했습니다.

조건	설명	목적
평점	6.0 이상 (로튼 6점대 이상)	일정 수준 이상의 기본적인 작품성 확보
평점 상위	사분위수 90% 이상	상위 10%에 해당하는 매우 높은 평점
투표수 (대중성)	평균 이상	투표 조작/샘플 부족 방지를 위한 최소한의 관심 확보
투표수 (낮은 대중성)	하위 50%	투표수가 적어 아직 대중적으로 덜 알려진 작품 선별

Sheets로 내보내기

분석 결과 설명
이 조건을 모두 만족하는 작품 리스트를 **'숨은 명작 TOP 20'**으로 제시했습니다.

시각화 활용: **투표수-평점 산점도 (06_vote_correlation.jpg)**에서 이 작품들이 어디에 위치하는지 시각적으로 보여주어, '투표수는 낮지만 평점은 높은 영역'에 집중되어 있음을 강조합니다.

💡 핵심 인사이트
"이 분석은 '좋은 작품이지만 아직 빛을 보지 못한' 작품들을 사용자에게 추천할 수 있는 강력한 근거를 제공합니다. OTT 플랫폼은 이 작품들을 발굴하여 큐레이션 서비스를 강화할 수 있습니다."

3. 📊 장르별 소비량 분석 (전체 콘텐츠 시장 트렌드)
분석 목적
전체 데이터셋(약 10,000개 작품)을 기준으로 콘텐츠 시장의 전반적인 장르 트렌드와 장르별 콘텐츠 공급 비중을 파악합니다.

분석 과정 및 결과 설명
장르별 작품 수 집계: 전체 작품 중 각 장르가 몇 개인지 집계합니다. (작품 하나가 여러 장르를 가질 수 있음)

장르별 비율 계산: 전체 작품 수 대비 각 장르의 비율(%)을 계산하여 **'소비량'**을 정의합니다.

주요 결과 강조 (테이블 활용):

**드라마 (45.22%)**와 **코미디 (33.13%)**가 압도적인 1, 2위를 차지했습니다.

상위 3개 장르(드라마, 코미디, 애니메이션)가 전체의 대부분을 차지함을 언급합니다.

💡 핵심 인사이트
"TMDB 데이터 기반 콘텐츠 시장은 드라마와 코미디 중심의 구성이 매우 강력합니다. 이는 콘텐츠 기획 시 가장 보편적인 흥행 공식은 드라마와 코미디임을 시사하며, 새로운 장르를 시도할 경우 틈새시장 공략 전략이 필요함을 보여줍니다."
