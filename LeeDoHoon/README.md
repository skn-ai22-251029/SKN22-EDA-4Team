# TMDB 데이터 EDA 분석 프로젝트

## 📋 프로젝트 개요

이 프로젝트는 TMDB(The Movie Database) 데이터를 활용하여 OTT 플랫폼별 작품 특성과 숨은 명작을 분석하는 탐색적 데이터 분석(EDA) 프로젝트입니다.

### 분석 목적

1. **OTT별 작품 필터링 기능** → Provider별 장르 강세 분석
2. **숨은 명작 찾기** → 투표수 낮고 평점 높은 작품 분석
   - 조건: 투표수는 평균 이상, 사분위수 90% 이상, 로튼 6점대 이상

### 데이터 출처

- **데이터 파일**: `data/tmdb_combined_10k.csv`
- **데이터 구성**: 총 7개 컬럼, 약 10,000개 행
- **주요 컬럼**: 
  - `id`: 작품 고유 ID
  - `title`: 작품 제목
  - `type`: 작품 유형 (movie/tv)
  - `genres`: 장르 (쉼표로 구분)
  - `providers`: OTT 플랫폼 (쉼표로 구분)
  - `vote_count`: 투표수
  - `vote_average`: 평균 평점

## 🔍 분석 내용

### 1. 프로젝트 개요
- 분석 목적 및 데이터 소개

### 2. 데이터 기본 정보 확인
- 행/열 수, 변수 타입, 결측치 확인
- 데이터 샘플 확인

### 3. 기술 통계 및 데이터 요약
- 수치형 변수: 평균, 중앙값, 표준편차, 최소/최대
- 범주형 변수: 각 범주의 빈도수 및 비율
- 분포 시각화 (히스토그램)

### 4. 결측치 및 이상치 탐색
- 결측치 확인 및 처리
- IQR 방법을 통한 이상치 탐색
- Boxplot 시각화
- **4.1 OTT별 작품 필터링 및 Provider별 장르 강세 분석**
  - 주요 OTT 플랫폼별 작품 수 분석
  - 플랫폼별 장르 강세 분석 및 시각화
  - 히트맵을 통한 플랫폼-장르 관계 분석

### 5. 변수 간 관계 분석
- 수치형 변수 간 상관관계 분석
- 산점도를 통한 투표수-평점 관계 시각화
- OTT 플랫폼별 평균 평점 비교

### 6. 파생 변수 생성 및 전처리 제안
- OTT 플랫폼 개수, 장르 개수 파생 변수
- 투표수 로그 변환
- 투표수/평점 구간 분류
- **6.1 숨은 명작 찾기 분석**
  - 분석 전략:
    1. 투표수는 평균 이상
    2. 사분위수 90% 이상 (평점 상위 10%)
    3. 로튼 6점대 이상 (평점 6.0 이상)
    4. 투표수는 하위 50% (대중성은 낮지만 평이 좋은 작품)
  - 숨은 명작 TOP 20 출력
  - 시각화 (산점도, 막대 그래프, 장르/OTT 분포)

### 7. 요약 및 인사이트 도출
- 주요 발견 사항 정리
- 분석 방향성 제시
- 최종 요약 테이블

## 📈 분석 인사이트

1. **OTT 플랫폼별 장르 강세를 파악**하여 각 플랫폼의 특성을 이해할 수 있음
2. **숨은 명작을 발견**하여 대중성은 낮지만 평이 좋은 작품을 추천할 수 있음
3. **투표수와 평점의 관계를 분석**하여 작품의 대중성과 평가의 균형을 확인할 수 있음
4. **각 OTT 플랫폼의 장르 포트폴리오를 비교**하여 차별화 전략을 수립할 수 있음
5. **숨은 명작 분석을 통해** 사용자에게 새로운 작품 추천 서비스를 제공할 수 있음
